# Earth-Shader
Rendering the earth using OpenGL and GLSL.
<br>
Features:<br>
0- Multi-texturing<br>
1- Diffuse and specular lighting.<br>
2- Physically-correct height map<br>
3- Highlights only on the water<br>
4- Real-time cloud animation.
